# selenium-webdriver-sample

# Setup
## selenium-server

install and start server.

```
% brew install selenium-server-standalone
% selenium-server
```

## Gems

```
% bundle install
```

or

```
% rbenv exec bundle install
```


## driver
install.

```
% ./setup.rb
```


# How to Run

```
% PATH=$PATH:. ./selenium.rb
```
